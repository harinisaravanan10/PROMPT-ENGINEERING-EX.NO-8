# INTRODUCTION

Text-to-image generation models are capable of creating realistic and artistic images from textual descriptions called prompts. The quality of generated images depends on how accurately the prompt describes the important features of the image such as objects, colors, textures, lighting, style, and composition.

This experiment demonstrates the process of analyzing an image, creating prompts, refining prompts, and reproducing similar images using AI image generation tools.

# OBJECTIVES

1. To understand prompt-based image generation.  
2. To analyze image elements such as color, lighting, and composition.  
3. To create effective prompts for image generation models.  
4. To compare generated images with original images.  
5. To understand the importance of prompt refinement.

# IMAGE 1 – SUNSET LANDSCAPE

## ORIGINAL IMAGE DESCRIPTION

The image shows a peaceful sunset landscape with mountains, a river, orange sky reflections, green trees, and soft clouds.

# STEP 1 – IMAGE ANALYSIS

| Feature | Observation |
|---|---|
| Objects | Mountains, river, trees, clouds |
| Colors | Orange, purple, blue, green |
| Lighting | Warm sunset lighting |
| Background | Natural outdoor environment |
| Style | Realistic digital art |
| Composition | Mountains in background, river in foreground |

# STEP 2 – BASIC PROMPT

```text
A landscape with mountains and a river during sunset.
```

# STEP 3 – REFINED PROMPT

```text
A peaceful sunset landscape with purple mountains, a calm river reflecting orange sunlight, green trees near the shore, soft clouds, and warm lighting.
```

# STEP 4 – STYLE REFINEMENT

```text
A realistic digital painting of a peaceful sunset landscape with purple mountains, a calm reflective river, green trees near the shoreline, soft orange clouds, cinematic lighting, and detailed natural scenery.
```

# GENERATED IMAGE OBSERVATION

| Aspect | Result |
|---|---|
| Color Matching | Very Good |
| Composition | Similar |
| Lighting | Accurate |
| Style | Realistic |
| Overall Similarity | High |

# IMAGE 2 – FUTURISTIC CITY

## ORIGINAL IMAGE DESCRIPTION

# STEP 1 – IMAGE ANALYSIS

| Feature | Observation |

| Colors | Blue, pink, purple neon |
| Lighting | Bright neon reflections |
| Background | Futuristic city environment |
| Style | Cyberpunk digital art |
| Composition | City streets with tall skyscrapers |

# STEP 2 – BASIC PROMPT

A highly detailed cyberpunk digital artwork of a futuristic city at night, featuring neon blue and pink lights, tall skyscrapers, flying cars, holographic advertisements, wet reflective roads, cinematic atmosphere, and realistic lighting effects.
